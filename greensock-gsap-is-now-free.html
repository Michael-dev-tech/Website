GreenSock’s GSAP Is Now Completely Free—Here’s Why It Matters

Posted: May 13, 2025

Big news for web animators and front-end developers: the GreenSock Animation Platform (GSAP), long regarded as the industry’s go-to toolkit for performant, cross-browser animations, has just gone fully open-source and free for all uses. Whether you’re crafting slick UI transitions, interactive infographics, or game-like experiences, GSAP’s powerful API is now unlocked without any license fees.

What Changed?

  • Full Open-Source Release: The core GSAP engine and all official plugins (ScrollTrigger, Draggable, MotionPath, etc.) have been relicensed under the permissive MIT license.
  • No More “Business Green” Plan: Previously, advanced features and plugins required a paid “Business Green” license. That paywall is gone—everything is free for personal, commercial, and enterprise use.
  • Community-Driven Roadmap: GreenSock will now accept proposals, pull requests, and plugin ideas directly from the community on GitHub, accelerating innovation and transparency.

Why It’s a Big Deal

GSAP has powered animations on countless high-profile sites and apps—its robustness, timing precision, and plugin ecosystem set it apart. Opening it up means:

  • Lower Barrier to Entry: Students, hobbyists, and startups can dive in without worrying about license costs.
  • Wider Adoption: Agencies and freelancers can include GSAP in client projects of any scale without legal headaches.
  • Faster Feature Development: Community contribution will drive new plugins, performance tweaks, and integrations at an even faster pace.

How to Get Started

  1. Visit the official GSAP GitHub repo and clone or download the code.
  2. Include GSAP in your project via npm: npm install gsap or directly via CDN: 
    <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/gsap.min.js"></script>
  3. Check out the updated docs and tutorials at greensock.com/docs.
  4. Explore sample projects and plugin demos in the GitHub repo’s `/examples` folder.

Community Reactions

Developers across Twitter and Dev.to are celebrating. From agencies that budgeted licensing fees to indie coders who’ve long championed open-source, the sentiment is clear: GSAP’s liberation has just leveled up the web animation game for everyone.

If you’ve never tried GSAP before, now’s the perfect time to experiment. And if you’re already a power user, consider contributing—whether it’s new plugin ideas, performance patches, or enhanced documentation.
